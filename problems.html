Selected Problems

2018 USAJMO Question 1

For each positive integer n, find the number of n-digit positive integers that satisfy both of the following conditions:

    no two consecutive digits are equal; and
    the last digit is a prime.

PuMaC 2008 A8 Modified

Given two sequences x(n) and y(n) defined by x(0) = y(0) = 7,

x(n) = 4x(n−1) + 3y(n−1), and

y(n) = 3y(n−1) + 2x(n−1),

Find lim n-> inf x(n)/y(n)

A problem I devised

Given positive integers a1,a2,...,an, find lim x-> 0 ((a1^x+a2^x+...+an^x)/n)^(1/x).
